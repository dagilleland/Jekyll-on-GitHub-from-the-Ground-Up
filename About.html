GitHub Metadata

When using GitHub Pages for your Jeykll website, you get access to a lot of metadata (basic information) about your GitHub repository. You access this in Jeykll by using the metadata name inside "mustache" or "handlebar" syntax to reference it like a variable. For example, the GitHub metadata for the repository name is site.github.repository_name, and you can get the value by typing &123;&123; site.github.repository_name &125;&125;. Here are the metadata values for this site:

  • &123;&123; site.github.build_revision &125;&125 – {{ site.github.build_revision }}
